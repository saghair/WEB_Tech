Quiz Questions and Answers

  1. What does HTML stand for?
    • A) HyperText Markup Language
    • B) Hyper Transfer Markup Language
    • C) Hyperlinks and Text Markup Language
    • D) Hyper Transfer Model Language
  2. What is the purpose of the <head> tag in HTML?
    • A) To define the header of the webpage
    • B) To contain metadata like title, character set, and link to stylesheets
    • C) To display the main content of the webpage
    • D) To define the layout section of the website
  3. What is an HTML editor used for?
    • A) To run HTML code
    • B) To design HTML pages
    • C) To edit and write HTML code
    • D) To debug HTML code
  4. Which of the following is the correct HTML element for inserting a line break?
    • A) <br>
    • B) <break>
    • C) <lb>
    • D) <hr>
  5. Which tag is used to create a hyperlink in HTML?
    • A) <link>
    • B) <a>
    • C) <href>
    • D) <curl>
  6. How do you create an unordered list in HTML?
    • A) <ol>
    • B) <ul>
    • C) <b>
    • D) <div>
  7. Which of the following is an HTML attribute?
    • A) <a>
    • B) class
    • C) <link>
    • D) style
  8. What is the default alignment of the <h1> heading in HTML?
    • A) Center
    • B) Left
    • C) Right
    • D) Justify
  9. Which tag is used to define a paragraph in HTML?
    • A) <p>
    • B) <pars>
    • C) <pr>
    • D) <paragraph>
  10. How do you add a comment in HTML?
    • A) <!-- comment -->
    • B) ' comment '
    • C) // comment
    • D) ** comment
  11. What is the correct HTML element to set a background color?
    • A) <Becolor>
    • B) <body bgcolor="color">
    • C) <background>
    • D) <style>
  12. Which HTML element is used for a quote or citation?
    • A) <quote>
    • B) <q>
    • C) <cite>
    • D) <blockquote>
  13. Which tag is used to insert an image in HTML?
    • A) <img>
    • B) <image>
    • C) <pic>
    • D) <graphic>
  14. What is a favicon in HTML?
    • A) A special image displayed in the browser's title bar
    • B) A background image for a webpage
    • C) A type of animation
    • D) A website's logo
  15. Which tag is used to define a table in HTML?
    • A) <table>
    • B) <tab>
    • C) <tr>
    • D) <td>
